## What's Inside

This repository hosts **4 fully interactive portals** that together simulate the complete digital ecosystem of a B2B enterprise distribution platform — built on Salesforce architecture principles.

| # | Portal | Description |
|---|---|---|
| 01 | **B2B Enterprise Portal** | Salesforce Experience Cloud-powered portal for enterprise buyers — RFQ submission, real-time pricing via API, quote lifecycle, delivery tracking, credit utilisation |
| 02 | **Dealer Portal** | Regional dealer interface — partner onboarding, order management, territory inventory, commission tracking |
| 03 | **Admin Console** | Internal operations dashboard — user & permission management, KYC queue, ERP sync monitoring, workflow configuration |
| 04 | **Salesforce CRM Console** | Sales Cloud implementation with 6 role-based personas (Sales Rep, Sales Manager, Sales Ops, Telecaller, Dealer RM, Admin) — Opportunity-to-Quote automation, approval workflows, forecasting |

## How to Run Locally

1. Clone or download this repository
   ```bash
   git clone https://github.com/chittaranjansahu/sf-portfolio.git
   ```
2. Open `index.html` in any modern browser
3. Click any portal card to open it in a new tab
4. For the B2B portal — login is pre-filled, just click **Log In**

> All files are self-contained. No server, build step, or dependencies required.
